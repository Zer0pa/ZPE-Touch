# Contributing

`zpe-touch` accepts focused contributions that preserve the frozen contact base and keep the bounded adopter boundaries explicit.

Before opening a pull request:

- keep contact, thermal, vibrotactile, and proprioceptive separated in code and claims
- keep affective touch out of scope unless the repo owner changes the publication boundary first
- add or update verification when changing the native extension, the local reference path, or proof artifacts
- do not remove adverse evidence to improve the public surface
