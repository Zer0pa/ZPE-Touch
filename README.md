# zpe-touch-codec

[![License: SAL v7.0](https://img.shields.io/badge/license-SAL%20v7.0-blue.svg)](LICENSE)

Geogram 3 Stage 1 Rust authoritative-path codec for bounded touch contact geometry.

Scope:

- contact trace
- receptor identity
- body region
- pressure profile

Out of scope for the authoritative path in this crate:

- thermal
- vibrotactile
- proprioceptive
- timing/frame
- RA-II helper side channels
